Hoe zet je PinterPal op voor je website?

Volg de stappen hieronder en start binnen enkele minuten met guided selling.

In 3 snelle stappen

  1. Plak de widget-root op je pagina: 
    <div id="pinterpal-widget-root"></div>
  2. Voeg de configuratie toe (je API-adres en tabel): 
    <script>
  window.PP_CONFIG = {
    API_BASE: "https://api.jouw-domein.com",
    API_USER: "admin",
    API_PASS: "secret",
    TABLE: "wijnen"
  };
</script>
  3. Laad het script net vóór </body>: 
    <script src="https://cdn.jouw-domein.com/pinterpal/app.js" defer></script>
  • 🔎 Persoonlijke aanbevelingen in real-time
  • 🧭 4–8 slimme vragen
  • 🛠️ Plug & play voor Shopify / Woo / Custom
  • 🛡️ GDPR-first · EU hosting

Tip: voeg ?open toe aan je URL om de widget direct te openen tijdens testen.
